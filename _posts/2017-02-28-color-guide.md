### Chart Fonts & Grid

* 폰트 사이즈 : 11px 
* 색상 #555555
* Outline : #55555
* Inline : #dddddd
<br>
<br>

### Table(Grid) Colors

* row hover : #eeeeee
* row selected : #DAECF5
<br>
<br>

### Chart Colors

1. Status chart color
  * Chart가 Status를 가질때
  
2. Basic chart color
  * Basic 하게 사용하는 색상 (UX팀에서 사용하는 Color)
  
3. Separation chart color
  * Chart에서 구분을 위하여 순차적으로 뿌려주는 색상 (Gantt, Trend Chart)
  
4. Basic bar chart color
  * 기본 Bar Chart (Bar Chart가 추가 될 경우 순차적으로 사용)
<br>
<br>
